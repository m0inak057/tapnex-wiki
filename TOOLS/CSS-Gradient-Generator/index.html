🎨 Understanding CSS Gradients

Linear Gradients:

  • Direction: Gradients flow in a straight line at a specified angle
  • 0°: Bottom to top (↑)
  • 90°: Left to right (→)
  • 180°: Top to bottom (↓)
  • 270°: Right to left (←)

Radial Gradients:

  • Gradients radiate outward from a central point
  • Perfect for spotlight effects and circular designs

💡 Gradient Design Tips

  • Color Harmony: Use colors that are close on the color wheel for smooth transitions
  • Contrast: Ensure sufficient contrast for text readability on gradients
  • Subtle is Better: Subtle gradients often look more professional than bold ones
  • Test on Different Screens: Gradients can look different on various displays
  • Consider Accessibility: Ensure text remains readable on gradient backgrounds
  • Multiple Color Stops: Add intermediate colors for more complex gradients
  • Animation: Gradients can be animated using CSS transitions or keyframes

💻 CSS Gradient Syntax

Basic Linear Gradient:

background: linear-gradient(90deg, #667eea, #764ba2);

With Multiple Colors:

background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);

Radial Gradient:

background: radial-gradient(circle, #667eea, #764ba2);

Gradient on Text:

background: linear-gradient(90deg, #667eea, #764ba2);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

🌐 Browser Support

  • Modern Browsers: Full support in Chrome, Firefox, Safari, Edge
  • Vendor Prefixes: Use prefixes for older browser versions
  • Fallback Colors: Always provide a solid color fallback
/* Fallback for older browsers */
background: #667eea;
/* Modern gradient */
background: linear-gradient(90deg, #667eea, #764ba2);
