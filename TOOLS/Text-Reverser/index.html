📋 How to Use

  1. Enter Text: Type or paste your text into the input box
  2. Choose Mode: Select whether to reverse characters, words, or lines
  3. Click Reverse: Press the "Reverse Text" button or hit Enter
  4. Copy Output: Click "Copy Output" to copy the reversed text
  5. Clear: Use the "Clear" button to reset both fields

💡 Reverse Modes Explained

1. Reverse Characters (Default):

  • Input: Hello World
  • Output: dlroW olleH
  • Every character is flipped, including spaces

2. Reverse Words:

  • Input: Hello World
  • Output: World Hello
  • Word order is reversed, characters within words stay the same

3. Reverse Lines:

  • Input: Line 1
    Line 2
    Line 3
  • Output: Line 3
    Line 2
    Line 1
  • Line order is reversed, content within lines stays the same

⚡ Keyboard Shortcuts

  • Enter: Reverse the text (when input is focused)
  • Ctrl/Cmd + Enter: Reverse text from anywhere
  • Escape: Clear both input and output
  • Ctrl/Cmd + C: Copy output (when output is focused)
  • Tab: Navigate between input and output fields

💻 How Text Reversal Works

Character Reversal Algorithm:

// Simple JavaScript implementation
function reverseText(text) {
 return text.split('').reverse().join('');
}

// Example:
reverseText("Hello"); // Returns: "olleH"

Word Reversal Algorithm:

// Reverse word order
function reverseWords(text) {
 return text.split(' ').reverse().join(' ');
}

// Example:
reverseWords("Hello World"); // Returns: "World Hello"
