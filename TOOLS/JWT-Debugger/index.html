What is a JWT?

 
A JSON Web Token (JWT) is a standard used to securely transmit information between parties as a JSON object. It consists of three parts separated by dots: the Header, the Payload, and the Signature. This tool decodes the Header and Payload, which are Base64URL-encoded JSON objects.

 
 
Frequently Asked Questions (FAQ)

 
Is this JWT Debugger secure?

 
Yes! This tool is 100% secure. All decoding happens in your browser using JavaScript. Your token is never sent to a server, so your sensitive information remains private.

 
 
Does this tool verify the signature?

 
No, this tool decodes the token to make it readable. It does not verify the signature, as that would require your private key or secret, which you should never paste into an online tool.
