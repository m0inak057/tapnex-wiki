How to Split a PDF File for Free

  1. Upload Your PDF: Click the "Upload" button or drag and drop your PDF file into the box.
  2. Choose Your Mode: Select how you want to split the file.
  3. Extract by Range: Choose this to get specific pages. Enter the page numbers or ranges you want (e.g., "1-3, 7, 9-11").
  4. Extract All Pages: Choose this to convert every page into its own separate PDF file.
  5. Split PDF: Click the "Split PDF" button to start the process.
  6. Download: Your new PDF(s) will be created instantly. A download will begin. (Note: If you extract all pages, they will be downloaded as a .zip file).

Frequently Asked Questions (FAQ)

Is this PDF splitter secure?

Yes! This tool is 100% secure. All processing is done in your browser using JavaScript. Your PDF file is never uploaded to a server, so your data remains private.

Is this tool really free?

Yes, our PDF splitter is completely free to use.

How do I enter a page range?

You can enter single pages (like 3), ranges (like 5-10), or a mix of both, separated by commas (like 1, 3-5, 12).

What happens if I "Extract All Pages"?

The tool will create a separate PDF document for every single page in your original file and package them into a single .zip file for you to download.
