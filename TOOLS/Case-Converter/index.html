üìö Case Types Explained

UPPERCASE (All Caps):

  • All letters are capitalized
  • Used for emphasis, headings, or constants in code
  • Example: THIS IS UPPERCASE TEXT

lowercase (All Small):

  • All letters are in lowercase
  • Used for email addresses, URLs, and casual writing
  • Example: this is lowercase text

Sentence case:

  • First letter of each sentence is capitalized
  • Standard format for regular writing and paragraphs
  • Example: This is sentence case. It looks natural.

Title Case (Proper Case):

  • First letter of each word is capitalized
  • Used for titles, headlines, and proper nouns
  • Example: This Is Title Case For Headings
