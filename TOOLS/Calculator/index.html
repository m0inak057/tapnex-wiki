Calculator

A modern, feature-rich calculator with keyboard support

✨ Features

  • Basic Operations: Addition, subtraction, multiplication, and division
  • Keyboard Support: Use your keyboard for faster calculations
  • Responsive Design: Works perfectly on all devices
  • Dark/Light Mode: Toggle between themes using the button in the top-right corner
  • Error Handling: Prevents division by zero
  • Smooth Animations: Beautiful transitions and hover effects

⌨️ Keyboard Shortcuts

  • Numbers (0-9): Input digits
  • +, -, *, /: Perform operations
  • Enter or =: Calculate result
  • C or Escape: Clear the calculator
  • Backspace: Delete last digit
  • . (period): Add decimal point
